# coronavirus
新冠疫情小区分布数据（数据来自各城市卫健委等官方微信公众号和官方网站）

演示网站：<a href="https://www.gubi.io/" target="_blank">https://www.gubi.io/</a>

1. 疫情场所的数据来源是什么？

数据来自各城市卫健委等官方微信公众号、官方网站、网友反馈等，可能会存在遗漏或非实时数据情况，请谅解。

2. 数据多久更新一次？

我们会不定期进行数据采集、整理工作，尽量及时更新数据。
